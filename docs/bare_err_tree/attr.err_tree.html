Internal constructor for a type derived with [err_tree][bare_err_tree::err_tree].

Call this to construct with the hidden [ErrTreePkg] field. Annotate any public constructor call with #[track_caller] so the error tracks the outer origin line.

§Example:

ⓘ
#[err_tree]
struct Foo { num: i32 }

impl Foo {
    #[track_caller]
    pub fn new(num: i32) -> Self {
        Foo::_tree(num)
    }
}

impl Error for Foo {}
impl Display for Foo {
    fn fmt(&self, f: &mut Formatter<'_>) -> fmt::Result {
        write!(f, "")
    }
}
